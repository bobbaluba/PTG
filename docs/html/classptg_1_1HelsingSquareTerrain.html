Detailed Description

Terrain generated using a modified version of the diamond square algorithm.

Solves some of the artifacts that occur in the diamond square algorithm, but introduces pointy tops The pointy tops can be removed by using a blurring filter as an intermediate or last step.

Complexity

Since this is just a minor modification of diamond square, the complexity is the same 

THETA(WIDTH^2)

Limitations and Artifacts

The size of the grid has to be 2^n+1 The points inside an area depends on all the points along the edges, this makes inconvenient to calculate just a portion of the heightMap.

In terms of artifacts and your preferences, the algorithm is a little better than diamond square. The ridges are gone, but mountain tops appear pinched.
